# 📝 Todo List App

A clean and responsive **Todo List Application** built with **HTML, CSS, and JavaScript**. This project allows users to manage their daily tasks with an intuitive interface and smooth interactions.

## ✨ Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* 💾 Save task updates
* ❌ Delete tasks
* ✅ Mark tasks as completed
* 🔍 Filter tasks (All, Active, Completed)
* 📱 Fully responsive design
* 🎨 Modern glassmorphism-inspired UI
* ⚡ Smooth animations and transitions

---

## 🚀 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 How to Run

1. Clone the repository.

```bash
git clone https://github.com/BOYWIDASTEEZ/tasklydailyapp.pyt
```

2. Navigate into the project folder.

```bash
cd todo-list
```

3. Open `index.html` in your browser.

No installation or dependencies are required.
